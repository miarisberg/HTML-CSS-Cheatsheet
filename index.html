HTML: Elements

The content of an HTML element is the info between the opening and closing tags of an element. Here are some examples:

Tag Name Description
<li> List item element Used to list items within:
  • Ordered list <ol>
  • Unordered list <ul>
<video> Video element Embeds media player for video playback.
src attribute will contain url to the video.
controls attribute will display controls for media player.
<div> Div element Container that divides an HTML doc. Can contain flow content like headings, paragraphs, link, etc.
<br> Break element Create a line break and does not need a closing tag.
<img> Image element Embeds image in documents. src attribute contains image url and is necessary
Does not have closing tag.
<body> Body element Content on HTML document and browswer will render the content within opening/closing tags.
<span> Span element Inline container to separate pieces of text.

HTML: Structure

The table below are some ways to structure an HTML document!

Tag Name Description
<a> Anchor element Used to create hyperlinks in an HTML doc. hrefcan point to URLs, files on the same server.
<head> Head element Contains info of HTML doc that isn't displayed as content aka metadata.
<html> HTML element Root of an HTML doc and should be added after !Doctype declaration.
<title> Title element Defines the title of HTML doc, and appears on browser tab.
Contained inside <head> element.

HTML: Semantic HTML

Semantic HTML tells developers what the code does and what content should go in it! Here are some examples.

Tag Name Description
<section> Section Defines elements in document such as chapters, headings, or other areas of the document with similar themes.
<article> Article Holds content that makes sense on its own.
<aside> Aside Marks additional info that can enhance another element, like a sidebar.
